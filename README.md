# Ingredient Analyzer

A web app that helps people understand food ingredients
in simple, everyday English using AI.

## Features
- Paste ingredients or food name
- Get a clear, non-technical explanation
- History sidebar to revisit past checks
- Expandable ingredient details

## Tech Stack
- Node.js
- Express
- JavaScript
- HTML / CSS
- AI API

## Note
This tool provides general information only.
It is not medical advice.
